<div align="center" style="font-family: 'Poppins', sans-serif; margin: 50px 0;">
  <h1 style="color: #007bff; font-size: 40px; font-weight: 700; margin-bottom: 10px;">INCACORE</h1>
  <h2 style="font-size: 22px; font-weight: 500; margin-top: 0; color: #555;">Eficiencia y Control Total para la Gestión Naval</h2>
  
  <p style="max-width: 800px; font-size: 16px; line-height: 1.7; text-align: center; color: #444; margin: 20px auto;">
    INCACORE es un sistema web que integra <strong style="color: #007bff;">gestión de mantenimiento</strong>, <strong style="color: #007bff;">control de inventarios</strong> y 
    <strong style="color: #007bff;">monitoreo operativo</strong> para la flota de Incamar.<br>
    Su objetivo es optimizar recursos, prevenir fallas y garantizar la disponibilidad operativa.
  </p>
</div>

## 🔧 Pasos para instalar el proyecto en forma local

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/ivanSegal/FooTalent-03.git
   cd FooTalent-03
   ```

2. Levantar los servicios Docker necesarios (incluyendo la base de datos en el puerto 5435):
   ```bash
   docker-compose up -d
   ```
   > **Nota:** La base de datos PostgreSQL se expone en el puerto local `5435`.

3. Ejecutar el backend con Maven:
   ```bash
   ./mvnw spring-boot:run
   ```

4. (Opcional) Ejecutar el frontend (si existe):
   ```bash
   cd frontend
   npm install
   npm start
   ```

5. Acceder a la aplicación en:
   - Backend: `http://localhost:8080` (o el puerto configurado)
   - Frontend: `http://localhost:3000` (o el puerto configurado)
